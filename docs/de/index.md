subtitle: Signature Check Plugin Dokumentation
---
Diese Dokumentation beschreibt die Verwendung vom scm-signature-check-plugin.
Sie steht in verschiedenen Sprachen und Versionen zur Verfügung, die im Menü rechts ausgewählt werden können.

Mit dem Signature Check Plugin kann man sicherstellen, dass nur signierte Changesets auf ein Repository gepushed werden können.
Dadurch kann die Integrität und Herkunft aller Changesets sichergestellt werden.
Bislang wird dieses Feature ausschließlich für Git Repositorys unterstützt.
